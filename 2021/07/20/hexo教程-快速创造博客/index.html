hexo教程-快速创造博客

2021-07-20- 2021-07-25
-
博客-博客hexo

hexo是什么

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

说白了他就是一个可以让你快速制作博客的模板

官网:HEXO

安装node.js和Git

由于hexo是基于node.js写的，所以你需要安装一个node.js

  1. 进入官网前去下载

2.选择左侧长期支持版并下载

3.开始安装

选安装路径

一直按next就行

在安装完成后打开命令行（windows键加R，输入cmd，回车）

输入

node -v

如果输出成功，会出现

v 版本号
比如 v14.17.1

如果报错，请检查自己的环境变量或是否安装完成

详细安装教程

接下来安装git

点击git官网

点击windows后会自动下载，因为一些原因，下载的很慢，要等一会

下载完成后点开

第一面点Next

选择自己的安装目录

这个选 Use Notepad as git’s default editor

其他的Next和Install 就行

安装好后打开命令行

输入

git --version

如果结果是 

git version 2.32.0.windows.2

那就行了

配置node.js和安装hexo

仍然打开命令行

输入一下命令

npm config set registry https://registry.npm.taobao.org
npm install -g cnpm - -registry=https://registry.npm.taobao.org
cnpm install hexo-cli -g

第一行意思是配置国内镜像（淘宝镜像）

第二行是下载cnpm

第三个是下载hexo

使用hexo

输入一下命令（在你要创建的地方在上面路径处输入cmd）

hexo init 文件夹名字
cd 你的文件夹名字
cnpm install

此时已经好了

但你要配置信息

进入 你的文件夹/_config.yml（用记事本打开）

有几个必填的

name为你的博客名字

subtitle为副标题

author为作者名字

language 输入zh-CN

tips:如何设置?

名字:值（比如: title:我）

你只需要写上值就行了

写你的第一篇文章

在你的文件夹下输入命令

hexo new 文章名字

此时他会在你的 文件夹\source_posts\下生成一个 文件夹.md

我建议下一个Typora进行编写

使用方法请自行寻找

编写玩后保存

在你的网站文件夹下输入命令

hexo s
或者
hexo server

启动后会提示

INFO  Start processing
INFO  Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

复制 http://localhost:4000 并打开此网站

此时你有了个自己的博客!!

Ctrl+C结束运行

上传至github

现在这个博客只有你能进入

所以你需要一些方法让别人也能看见

正常的网站需要买域名加服务器

所幸github可以让你免费运行

首先登入官网

点击右上角 Sign up 进行注册

你已经习惯了国内的注册要手机号了吧

但他不需要

他需要你的邮箱

你可以借助（百度）翻译完成注册

注册好后点击左边的New

设置如下

点击Create repository

回到你的博客文件夹

修改_config.yml如下

deploy:
  type: git
  repo:  https://github.com/你的github名字/你的github名字.github.io.git
  branch: main

之后打开命令行输入:

hexo d
hexo g

如果有“Username for”和”Password for” 请输入你的github用户名（username）和密码（password）

让后你可以访问 你的github名字.github.io
